Detailed Description

Pojedyncza siatka (mesh) modelu: geometria + tekstury + obiekty OpenGL.

Mesh przechowuje wierzchołki i indeksy oraz konfiguruje VAO/VBO/EBO. Renderowanie odbywa się przez Draw().
