Detailed Description

Statyczny system kolizji toru: budowanie ścian z polilinii i test kolizji okrąg–odcinek.

System przechowuje dwie surowe polilinie (lewa i prawa strona toru), a następnie buduje z nich listę odcinków (walls) wykorzystywanych do testów kolizji.

Kolizja jest liczona w 2D w płaszczyźnie XZ (pozycja auta: carPos.x i carPos.z).
