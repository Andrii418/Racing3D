Detailed Description

Kamera sceny 3D (widok + projekcja) z trybem „chase” i trybem pierwszoosobowym.

Klasa utrzymuje pozycję i orientację kamery (Position, Front, Up) oraz parametry Eulera (Yaw, Pitch). Udostępnia macierz widoku (GetViewMatrix) i macierz projekcji perspektywicznej (GetProjectionMatrix).

Dodatkowo implementuje:

  • kamerę podążającą za pojazdem (trzecia osoba) przez FollowCar
  • kamerę „z przodu” (pierwsza osoba / zderzak) przez SetFrontCamera
