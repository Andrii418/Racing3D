Detailed Description

Struktura przechowująca dane geometryczne (siatkę) pojedynczej części modelu samochodu.

Struktura zawiera dane wierzchołków, normalnych, współrzędnych UV, indeksów oraz uchwyty buforów OpenGL (VAO/VBO/EBO) niezbędne do renderowania metodą glDrawElements.
