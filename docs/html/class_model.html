Detailed Description

Model 3D składający się z wielu siatek wczytanych przez Assimp.

Model wczytuje scenę z pliku (np. OBJ), iteruje po węzłach i siatkach Assimp, buduje obiekty Mesh oraz ładuje tekstury materiałów (z cache).
