Detailed Description

Prosty tor w formie płaskiej siatki trójkątów generowanej proceduralnie.

Klasa tworzy siatkę o zadanej rozdzielczości (gridSize) w zakresie [-0.5, 0.5] w osiach X/Z, a następnie skaluje ją macierzą ModelMatrix. Geometria zawiera pozycję i normalną dla każdego wierzchołka (układ interleaved: 3 floaty pozycji + 3 floaty normalnej).
